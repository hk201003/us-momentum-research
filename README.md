# US Momentum Research

米国株モメンタムファンドの仕組み調査と独自バックテスト

## 背景

2025年12月、三井住友トラスト・アセットマネジメントが「SMT 米国株式モメンタムファンド（愛称：トレンドランキング・米国株）」を設定。米日欧中4市場をカバーする「SMTトレンドランキングシリーズ」の一つ。

しかし設定から間もないため実績データがほぼない。そこで **公開されたルールに基づいてバックテストを実行し、実力を検証する。**

## SMTファンドのルール

| 項目 | 内容 |
|------|------|
| ユニバース | 時価総額上位 約500銘柄 |
| 短期枠 | 6ヶ月リターン上位7銘柄 |
| 中期枠 | 12ヶ月リターン上位7銘柄 |
| 長期枠 | 36ヶ月リターン上位7銘柄 |
| 重複ルール | 短期優先、重複銘柄は次点に繰り下げ |
| 配分 | 21銘柄均等配分 |
| リバランス | 年4回（2/5/8/11月末基準） |
| 信託報酬 | 年0.77%（税込） |
| 購入手数料 | 最大3.3%（税込） |
| NISA | 成長投資枠対象 |

### 注意事項

- **生存バイアスあり**: 現在のS&P500構成銘柄で過去を計算しているため、実際よりやや良く出ている
- **コスト未反映**: 信託報酬0.77%、購入手数料最大3.3%、売買コスト未控除。実際はこれより数%悪い
- **トレンドフィルターなし**: MDD -37.7%は暴落をまともに食らった結果

## 構造的弱点の分析

**SPYは大幅に上回っているが、いくつかの構造的弱点がある:**

1. **MDDが深い（-37.7%）** — トレンドフィルターがないため暴落時の防御がない
2. **均等配分** — 高ボラ銘柄も低ボラ銘柄も同額。リスク寄与が偏る
3. **銘柄単位の選定** — 個別銘柄の過去リターンのみで選定。セクター分散の仕組みがない
4. **年4回リバランス** — モメンタムの効果は時間とともに減衰するため、頻度が低い可能性
5. **ボラティリティが高い（27.6%）** — SPYの1.6倍。リスク調整後で見ると超過リターンは縮小する

## 改良版バックテスト

4つの弱点に対応する改良を加え、個別効果と全部入りの効果を検証。

### 改良内容

| 改良 | 内容 | 狙い |
|------|------|------|
| トレンドフィルター | SPY >= SMA200 → フル投資 / < SMA200 → 50%縮小 | 暴落時の防御 |
| 逆ボラティリティ加重 | (1/vol_60d) で配分 | リスク寄与の均等化 |
| 月次リバランス | 年4回 → 毎月末 | モメンタム減衰への対応 |
| セクター分散キャップ | 1セクター最大3銘柄 | 特定セクター集中の回避 |

### 改良版の評価

**最も効果が大きかったのは月次リバランス**（Sharpe 1.16→1.32）。モメンタムの鮮度が重要。

**全部入り（All Improved）は守備力に優れる:**
- MDD: -37.7% → **-29.6%**（8ポイント改善）
- Vol: 27.6% → **20.0%**（7.6ポイント低下）
- COVID-19時: MDD -37.7% → **-24.2%**（13ポイント改善）

**トレードオフ:** CAGR は 32.0% → 25.7% に低下。トレンドフィルターで暴落を回避する代わりに、回復局面のリターンも一部失う。

## パラメータ感度分析

### ルックバック期間（21組み合わせ）

Short(3M/6M/9M) × Mid(6M/12M/18M) × Long(24M/36M/48M) の全組み合わせを検証。

- **Long=24Mが最も安定**: Sharpe 1.25〜1.33（SMTデフォルトの36Mは5位/21）
- **全体のSharpe標準偏差=0.08** → パラメータに対してロバスト
- Short期間の影響は小さい（avg Sharpe 1.26〜1.30）

### リバランス頻度（5種類）

| 頻度 | 回/年 | Sharpe | CAGR | MDD |
|------|:-----:|:------:|:----:|:---:|
| Quarterly | 4 | 1.16 | +32.0% | -37.7% |
| Bimonthly | 6 | **1.28** | +34.9% | -35.7% |
| Monthly | 12 | **1.28** | +25.8% | -27.9% |
| Biweekly | 25 | 1.28 | +28.9% | -33.3% |
| Weekly | 51 | 1.21 | +22.6% | -27.6% |

逆U字型: 隔月〜月次が最適、週次は取引コスト増で劣化。

### 高度なバリアント（10種類）

1Mスキップ、銘柄数変更（15/21/30）、バケットティルト、リスク調整モメンタム、単一期間モメンタム等を検証。

**結果: ベースライン（7/7/7）が最良（Sharpe 1.28）。どのバリアントも改善せず。**

## 生存バイアスの定量化

2段階で検証。

### Step 1: 追加日のみ考慮（Hist-v1）

現在のS&P500構成銘柄の追加日をWikipediaから取得し、各銘柄が追加された日以降のみ投資対象とする。

### Step 2: 追加+除外を完全追跡（True-Historical）★最終版

WikipediaのS&P500変更履歴（389イベント）を使い、**各時点の実際のS&P500構成銘柄を復元**。
過去にS&P500にいたが除外された銘柄（買収・業績悪化等）も在籍期間中は投資対象に含める。
843銘柄をダウンロード（現在503 + 過去に除外された340）。

### 3段階比較

| Strategy | Sharpe | CAGR | MDD | Vol |
|----------|:------:|:----:|:---:|:---:|
| Naive Original | 1.16 | +32.0% | -37.7% | 27.6% |
| Hist-v1 Original | 0.71 | +17.5% | -35.3% | 24.5% |
| **True-Hist Original** | **0.68** | **+16.9%** | -36.8% | 24.8% |
| Naive Improved | 1.28 | +25.8% | -27.9% | 20.2% |
| Hist-v1 Improved | 0.93 | +16.5% | -23.9% | 17.7% |
| **True-Hist Improved** | **0.83** | **+15.2%** | -25.2% | 18.3% |
| SPY | 0.79 | +13.5% | -33.7% | 17.2% |

### バイアスの内訳

| バイアス要因 | Original (Sharpe) | Improved (Sharpe) |
|-------------|:---:|:---:|
| **追加日バイアス**（未来のS&P500入りを予知） | +0.44 | +0.35 |
| **除外銘柄バイアス**（負け組が消える） | +0.03 | +0.10 |
| **合計** | **+0.48** | **+0.44** |

→ バイアスの90%以上は「追加日バイアス」。除外銘柄の影響は比較的小さい。

### 除外された銘柄の実態

BT期間中にS&P500から除外された109銘柄のリターン:
- **61%が負のリターン**、中央値 -30%
- ワースト: COL(-97%), LUMN(-86%), COTY(-85%), ENPH(-84%)
- これらの「負け組」がNaiveバックテストでは最初から除外されている

### 主な発見

1. **Naive Sharpe 1.28 → True-Historical Sharpe 0.83**: バイアスは+0.44
2. **True-Hist Improved はSPYをわずかに上回る**: Sharpe 0.83 vs 0.79、CAGR +15.2% vs +13.5%
3. **取引コスト考慮後（年1-2%）**: Sharpe 0.72〜0.78、CAGR +13.2%〜+14.2%
4. **SPYとの差はほぼゼロ**: コスト後は +0.7%/年 〜 -0.3%/年
5. **トレンドフィルターの価値は残る**: MDD -25.2%（True-Hist Improved）vs -36.8%（True-Hist Original）

### 結論: モメンタム戦略の真の実力

**バックテスト → 実運用の乖離:**

| 要因 | Sharpe影響 |
|------|:---:|
| 生存バイアス（追加日+除外銘柄） | -0.44 |
| 取引コスト（年1-2%） | -0.05〜0.10 |
| 執行スリッページ | -0.02〜0.05 |
| **合計** | **-0.51〜0.59** |

→ **モメンタム戦略の実運用Sharpe: 0.7〜0.8程度**（SPYとほぼ同等）

**一言でいうと: モメンタム戦略は「SPYに対してほとんどαがない」が真実。バックテストのSharpe 1.16〜1.28は幻。**

## 実行方法

```bash
pip install yfinance pandas numpy

# 改良版バックテスト（5パターン比較）
python backtest_smt_improved.py

# 生存バイアス除去バックテスト（最終版: 追加+除外完全追跡）
python backtest_true_historical.py

python backtest_period_sensitivity.py     # パラメータ感度

```
