# US Momentum Research

米国株モメンタムファンドの仕組み調査と独自バックテスト

## 背景

2025年12月、三井住友トラスト・アセットマネジメントが「SMT 米国株式モメンタムファンド（愛称：トレンドランキング・米国株）」を設定。米日欧中4市場をカバーする「SMTトレンドランキングシリーズ」の一つ。

しかし設定から間もないため実績データがほぼない。そこで **公開されたルールに基づいてバックテストを実行し、実力を検証する。**

## SMTファンドのルール

| 項目 | 内容 |
|------|------|
| ユニバース | 時価総額上位 約500銘柄 |
| 短期枠 | 6ヶ月リターン上位7銘柄 |
| 中期枠 | 12ヶ月リターン上位7銘柄 |
| 長期枠 | 36ヶ月リターン上位7銘柄 |
| 重複ルール | 短期優先、重複銘柄は次点に繰り下げ |
| 配分 | 21銘柄均等配分 |
| リバランス | 年4回（2/5/8/11月末基準） |
| 信託報酬 | 年0.77%（税込） |
| 購入手数料 | 最大3.3%（税込） |
| NISA | 成長投資枠対象 |

## バックテスト結果（2011-2026年）

S&P500構成銘柄で、上記ルールを再現してバックテストを実行。

### 注意事項

- **生存バイアスあり**: 現在のS&P500構成銘柄で過去を計算しているため、実際よりやや良く出ている
- **コスト未反映**: 信託報酬0.77%、購入手数料最大3.3%、売買コスト未控除。実際はこれより数%悪い
- **トレンドフィルターなし**: MDD -37.7%は暴落をまともに食らった結果

## 評価

**SPYは大幅に上回っているが、いくつかの構造的弱点がある:**

1. **MDDが深い（-37.7%）** — トレンドフィルターがないため暴落時の防御がない
2. **均等配分** — 高ボラ銘柄も低ボラ銘柄も同額。リスク寄与が偏る
3. **銘柄単位の選定** — 個別銘柄の過去リターンのみで選定。セクター分散の仕組みがない
4. **年4回リバランス** — モメンタムの効果は時間とともに減衰するため、頻度が低い可能性
5. **ボラティリティが高い（27.6%）** — SPYの1.6倍。リスク調整後で見ると超過リターンは縮小する
