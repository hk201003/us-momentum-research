# US Momentum Research

米国株モメンタムファンドの仕組み調査と独自バックテスト

## 背景

2025年12月、三井住友トラスト・アセットマネジメントが「SMT 米国株式モメンタムファンド（愛称：トレンドランキング・米国株）」を設定。米日欧中4市場をカバーする「SMTトレンドランキングシリーズ」の一つ。

しかし設定から間もないため実績データがほぼない。そこで **公開されたルールに基づいてバックテストを実行し、実力を検証する。**

## SMTファンドのルール

| 項目 | 内容 |
|------|------|
| ユニバース | 時価総額上位 約500銘柄 |
| 短期枠 | 6ヶ月リターン上位7銘柄 |
| 中期枠 | 12ヶ月リターン上位7銘柄 |
| 長期枠 | 36ヶ月リターン上位7銘柄 |
| 重複ルール | 短期優先、重複銘柄は次点に繰り下げ |
| 配分 | 21銘柄均等配分 |
| リバランス | 年4回（2/5/8/11月末基準） |
| 信託報酬 | 年0.77%（税込） |
| 購入手数料 | 最大3.3%（税込） |
| NISA | 成長投資枠対象 |

### 注意事項

- **生存バイアスあり**: 現在のS&P500構成銘柄で過去を計算しているため、実際よりやや良く出ている
- **コスト未反映**: 信託報酬0.77%、購入手数料最大3.3%、売買コスト未控除。実際はこれより数%悪い
- **トレンドフィルターなし**: MDD -37.7%は暴落をまともに食らった結果

## 構造的弱点の分析

**SPYは大幅に上回っているが、いくつかの構造的弱点がある:**

1. **MDDが深い（-37.7%）** — トレンドフィルターがないため暴落時の防御がない
2. **均等配分** — 高ボラ銘柄も低ボラ銘柄も同額。リスク寄与が偏る
3. **銘柄単位の選定** — 個別銘柄の過去リターンのみで選定。セクター分散の仕組みがない
4. **年4回リバランス** — モメンタムの効果は時間とともに減衰するため、頻度が低い可能性
5. **ボラティリティが高い（27.6%）** — SPYの1.6倍。リスク調整後で見ると超過リターンは縮小する

## 改良版バックテスト

4つの弱点に対応する改良を加え、個別効果と全部入りの効果を検証。

### 改良内容

| 改良 | 内容 | 狙い |
|------|------|------|
| トレンドフィルター | SPY >= SMA200 → フル投資 / < SMA200 → 50%縮小 | 暴落時の防御 |
| 逆ボラティリティ加重 | (1/vol_60d) で配分 | リスク寄与の均等化 |
| 月次リバランス | 年4回 → 毎月末 | モメンタム減衰への対応 |
| セクター分散キャップ | 1セクター最大3銘柄 | 特定セクター集中の回避 |

### 改良版の評価

**最も効果が大きかったのは月次リバランス**（Sharpe 1.16→1.32）。モメンタムの鮮度が重要。

**全部入り（All Improved）は守備力に優れる:**
- MDD: -37.7% → **-29.6%**（8ポイント改善）
- Vol: 27.6% → **20.0%**（7.6ポイント低下）
- COVID-19時: MDD -37.7% → **-24.2%**（13ポイント改善）

**トレードオフ:** CAGR は 32.0% → 25.7% に低下。トレンドフィルターで暴落を回避する代わりに、回復局面のリターンも一部失う。

## 実行方法

```bash
pip install yfinance pandas numpy

# SMTオリジナルの再現バックテスト
python backtest_smt_us_momentum.py

# 改良版バックテスト（5パターン比較）
python backtest_smt_improved.py
```

> yfinanceでS&P500構成銘柄の株価データをダウンロードするため、初回実行に数分かかります。
